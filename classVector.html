Detailed Description

template<typename T, std::size_t N>
class Vector< T, N >

A templated class representing a vector of arbitrary size and type.

This class provides functionality for creating and manipulating vectors of any type T and size N. It supports common operations like addition, subtraction, scalar multiplication, dot product, and cross product (for 2D vectors).

Template Parameters
TThe type of the elements in the vector (e.g., int, float).
NThe number of dimensions of the vector (size of the vector).
