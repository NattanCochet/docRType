Parameterized constructor for the Drawable class.

Initializes the Drawable object with specific values for the path, sprite sheet rectangle, animation, scale, player status, and frame data for each direction (up, down, left, right).

Parameters
typeThe type of the drawable object (e.g., "player", "enemy").
isAnimatedA flag indicating if the sprite is animated.
scaleThe scale of the sprite.
defaultFrameFrame data for the default (static) animation.
secondForDefaultFrameDuration for each frame of the default animation.
frameUPFrame data for the "up" animation.
secondForFrameUpDuration for each frame of the "up" animation.
frameDownFrame data for the "down" animation.
secondForFrameDownDuration for each frame of the "down" animation.
frameLeftFrame data for the "left" animation.
secondForFrameLeftDuration for each frame of the "left" animation.
frameRightFrame data for the "right" animation.
secondForFrameRightDuration for each frame of the "right" animation.
