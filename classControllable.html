Detailed Description

A class representing an entity that can be controlled in the system.

Base class representing objects that can be controlled.

The Controllable class holds the client ID and provides methods to get and set it. This class can be used for controlling entities like players or other interactive objects.

This class serves as a base class for objects that can be controlled within the game, though it does not define specific control mechanisms. It can be extended by other classes that define more concrete control behavior.
