Detailed Description

template<typename Component>
class ComponentArray< Component >

A dynamic array that manages components, with optional values for each slot.

Template class for managing an array of optional components.

This template class is designed to manage an array of components, supporting dynamic resizing, insertion, and erasure while utilizing std::optional to handle empty slots.

Template Parameters
ComponentThe type of components managed by the array.
ComponentThe type of component to store in the array
