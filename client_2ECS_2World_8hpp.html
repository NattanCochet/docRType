Detailed Description

Header file for the World class, which manages the game world and handles key and mouse actions.

The World class is responsible for the core game state, including the management of entities through the Registry, handling user input (keyboard and mouse), and providing access to game assets like textures and fonts. It also manages the game clock and the state of the game (e.g., menu, in-game, etc.).
