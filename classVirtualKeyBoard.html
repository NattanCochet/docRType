Detailed Description

Represents a virtual keyboard for controlling game actions.

The VirtualKeyBoard class manages a set of control keys and their states (pressed or not). It provides methods to modify and check the state of the controls (e.g., UP, DOWN, LEFT, RIGHT, SHOOT, FORCE, START).

enum VirtualKeyBoard::CONTROL

Enum for defining the control keys available on the virtual keyboard.

The CONTROL enum defines the various control keys that can be mapped to actions in the game, such as movement directions or triggering actions.
