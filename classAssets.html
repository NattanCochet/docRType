Detailed Description

Manages loading and accessing game assets like sprites.

Class to load and manage sprites, text, and rectangles.

This class allows for loading sprites from files and storing them in a map.

This class handles the loading of different graphical assets (sprites, texts, rectangles) and provides access methods to retrieve them based on their file paths or names.
