Detailed Description

Represents an enemy in the game.

The Ennemy class holds an ID and a pointer to a bot (ABot) that represents the enemy's behavior. The bot could be a specific type of bot such as a LinearShooter or TpShooter, and the ID can be used for unique identification of the enemy.
