Detailed Description

Registry class for managing entities, components, and systems.

A class that manages entities and their components, as well as systems that operate on them.

This class serves as the main container and manager for the ECS (Entity Component System). It handles entity creation/deletion, component management, and system registration.

This class handles the registration, retrieval, and management of entities, components, and systems in the game world. It provides functionality to register and store components for entities, spawn and kill entities, and register systems for game logic.
