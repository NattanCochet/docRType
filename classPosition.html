Detailed Description

A class to represent and manage the position of an object.

Class to manage 2D positions in the game.

This class manages the position of an object using two sf::Vector2f objects: one for the current position and one for the default position. It provides methods to get and set the position values.

This class represents a 2D position using sf::Vector2f. It allows storing the current position and a default position, as well as modifying them.
